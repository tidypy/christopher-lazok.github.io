# Project: IT Consulting Hero Section

## Project Goal

To create a unique, modern, and customized hero section to be embedded in a Google Site. The
section will showcase IT capabilities with a focus on business intelligence and medium business
SaaS solutions. It will feature animations and be optimized for SEO.

## Technology Stack

*   **Framework:** Next.js (with React)
*   **Language:** TypeScript
*   **UI Components:** Custom (Glassmorphism Style)
*   **Styling:** Tailwind CSS
*   **Animations:** Framer Motion
*   **Background:** Vanta.js (Three.js)
*   **Development Environment:** Node.js
